This is a stardew Valley mod that allows you to change sleeve color, suggested by Orphea.
Open your inventory and at your inventory page there will be a box with 3 sliders. Top slider is red, next is green, and next blue.
Each sleeve color is unique to that shirt.

This mod saves data in a text file! If you want to remove data.. (not reccomended)

  Open the file (create a dupe incase you mess up)
  
  The data is stored the following way
  
    SHIRT_ID : R : G : B
    
  Remove the shirt ID line that you want.
  
  Adding more characters will mess up the file

Have fun!
